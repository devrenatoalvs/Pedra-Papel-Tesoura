# 🪨📄✂️ Jogo Pedra, Papel e Tesoura

Este repositório contém um jogo simples de **Pedra, Papel e Tesoura**, desenvolvido em **Python**, como aplicação prática dos conceitos estudados em aula.

O jogo é executado no **terminal** e permite que o usuário jogue contra o computador, aplicando lógica condicional e interação via entrada de dados.

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é praticar e consolidar conceitos básicos de programação, como:

- Estruturas condicionais (`if`, `elif`, `else`)
- Entrada e saída de dados (`input`, `print`)
- Uso de variáveis
- Lógica de decisão
- Conceitos iniciais de versionamento com **Git e GitHub**

---

## 🕹️ Como o jogo funciona

1. O jogador escolhe uma das opções:
   - Pedra
   - Papel
   - Tesoura
2. O computador escolhe uma opção aleatoriamente
3. O programa compara as escolhas
4. O resultado da rodada é exibido no terminal:
   - Vitória do jogador
   - Vitória do computador
   - Empate

---

## 🛠️ Tecnologias utilizadas

- **Python 3**

- **GitHub** para hospedagem do repositório
